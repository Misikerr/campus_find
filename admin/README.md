# Admin Panel Setup

1.  **Database Setup**:
    Open your browser and navigate to:
    `http://localhost/Compus Finder/admin/setup.php`
    
    This will create the necessary tables (`admins`, `categories`, `locations`, `announcements`, `item_reports`) and add the default admin user.

2.  **Login**:
    Navigate to:
    `http://localhost/Compus Finder/admin/`
    
    **Default Credentials:**
    - Username: ``
    - Password: ``

3.  **Features**:
    - **Dashboard**: Overview of stats.
    - **Manage Posts**: Approve, Reject, Delete, and Resolve lost/found items.
    - **Reports**: Handle flagged items.
    - **Users**: Manage admin accounts.
    - **Categories & Locations**: Manage the dropdown options dynamically.
    - **Announcements**: Create site-wide announcements.
